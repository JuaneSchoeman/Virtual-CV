# Virtual CV — Personal Portfolio Website

A fully custom portfolio website built from scratch to showcase my skills, projects, and experience as a software developer.

## Live site
🔗 https://juaneschoeman.github.io/Virtual-CV/

## Features
- Responsive single-page design with smooth section navigation
- Projects page that pulls live data from the GitHub API
- Downloadable CV
- Clean, minimal aesthetic built without any CSS framework

## Tech stack
- HTML5, CSS3, JavaScript (Vanilla)
- GitHub Pages for hosting
- GitHub REST API for dynamic project display

## Why I built it
Instead of a static PDF CV, I wanted something that updates automatically as I push new projects to GitHub — and that shows off front-end skills at the same time.
